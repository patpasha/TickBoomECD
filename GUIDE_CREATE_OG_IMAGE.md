# 🎨 Guide : Créer og-image.png pour TickBoom

## 📐 Spécifications Techniques

**Dimensions** : 1200 × 630 pixels (ratio 1.91:1)
**Format** : PNG ou JPG
**Poids** : < 1 MB (idéal < 300 KB)
**Nom** : `og-image.png`
**Emplacement** : `/app/frontend/public/og-image.png`

---

## 🎨 Styles à Respecter

### Couleurs (Cyberpunk Theme)
- **Fond** : #0a0a0f (noir profond)
- **Texte principal** : #ffffff (blanc)
- **Accents** : 
  - Cyan : #00f0ff
  - Purple : #b537ff
  - Pink : #ff006e
  - Yellow : #ffd60a
- **Gradients** : from-[#00f0ff] via-[#b537ff] to-[#ff006e]

### Typographie
- **Font** : JetBrains Mono (ou monospace similaire)
- **Tagline** : Bold, 72-80px
- **Subtitle** : Regular, 36-42px
- **Badges** : Medium, 24-28px

### Effets
- Glow effect sur texte principal (cyan/purple)
- Border gradient sur screenshot
- Backdrop blur sur badges
- Shadows pour depth

## 🛠️ Outils pour Créer

### Option 1 : Canva (Le plus simple) 🌟 RECOMMANDÉ
1. Aller sur https://www.canva.com
2. Créer un design "Custom" 1200×630
3. Fond noir (#0a0a0f)
4. Ajouter texte "The Do or Don't App" (gradient)
5. Importer screenshot iPhone
6. Ajouter logo TickBoom
7. Exporter en PNG

**Template Canva** :
- Chercher "Open Graph Image"
- Modifier avec vos éléments

### Option 2 : Figma (Plus de contrôle)
1. Nouveau fichier 1200×630
2. Rectangle fond noir
3. Importer assets
4. Appliquer gradients et effets
5. Exporter PNG 2x (haute qualité)

### Option 3 : Photoshop / GIMP
1. Nouveau document 1200×630, 72 DPI
2. Fond #0a0a0f
3. Ajouter calques pour chaque élément
4. Appliquer effets (glow, gradient)
5. Exporter pour web (PNG-24)

### Option 4 : En Ligne (Gratuit)
**Creatopy** : https://www.creatopy.com
**Snappa** : https://snappa.com
**Placeit** : https://placeit.net

## 📝 Note Importante

**L'og-image.png est déjà référencé dans votre SEO.jsx** :
```javascript
const ogImage = image || `${baseUrl}/og-image.png`;
```

Donc une fois le fichier créé et placé dans `/app/frontend/public/`, il sera automatiquement utilisé pour :
- Facebook Open Graph
- Twitter Cards
- LinkedIn previews
- Autres réseaux sociaux

**Aucune modification de code nécessaire !** 🎉
